# Witcher 3 Build Planner

A single-page build planner for **The Witcher 3: Wild Hunt (Next Gen / Patch 4.0)**. Plan your character build before spending skill points in-game.

## What you can do

- Browse and rank skills across the Combat, Signs, Alchemy, and General trees
- Assign skills to active slots and slot mutagens into slot groups
- Choose a Blood & Wine mutation
- See the minimum character level your build requires
- Share your build via a generated URL

## How to use

Open `index.html` directly in a browser — no install, no server, no build step required.

To load a shared build, paste the `?b=` URL into your browser's address bar or use the Import button on the page.

## Game version

All skill data reflects **Patch 4.0 (Next Gen Update)**. Skills max at Rank 3; descriptions and tier placements match the Witcher Wiki for that patch.
